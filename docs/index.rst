Contribute!
===========
**incomevis** is an open source software originally written by `Sang T.Truong <sttruong@stanford.edu>`_ and `Professor Humberto Barreto <hbarreto@depauw.edu>`_. The package released under the MIT licence.
It works on Windows, Mac, and Linux, with Python 2 or Python 3.
The code is hosted on `Github <https://github.com/sangttruong/incomevis>`_, where you can push improvements, report bugs and ask for help.
